# Tier 2 Support Program

## Overview

The Tier 2 Support Program was created to provide students and aspiring IT professionals with opportunities to gain hands-on experience using a live ticketing platform.

Tier 2 volunteers assist with ticket escalation, troubleshooting, documentation, and knowledge sharing while developing practical support experience.

## Objectives

* Provide real-world help desk experience
* Improve troubleshooting skills
* Encourage peer-to-peer learning
* Develop documentation habits
* Build resume-worthy experience

## Tier 1 Responsibilities

Tier 1 support agents are responsible for:

* Responding to incoming tickets
* Gathering information from users
* Searching the knowledge base
* Attempting initial troubleshooting
* Escalating unresolved issues

## Tier 2 Responsibilities

Tier 2 support volunteers are responsible for:

* Handling escalated tickets
* Performing advanced troubleshooting
* Creating and updating knowledge base articles
* Assisting Tier 1 agents
* Identifying recurring issues and solutions

## Escalation Process

1. User submits ticket
2. Tier 1 reviews and attempts resolution
3. Ticket is escalated if unresolved
4. Tier 2 investigates and documents findings
5. Resolution is provided to the user
6. Knowledge base is updated when appropriate

## Skills Developed

Participants gain experience in:

* Ticket management
* Customer communication
* Technical troubleshooting
* Documentation
* Escalation procedures
* Team collaboration

* Increase knowledge base coverage
