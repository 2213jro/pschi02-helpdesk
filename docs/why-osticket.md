# Why osTicket?

## Overview

The PSCHI02 Student Support Platform was built using osTicket as its primary help desk and ticket management system.

osTicket was selected because it is a widely adopted open-source support platform trusted by millions of users worldwide. It provides the tools and workflows commonly found in professional IT support environments while remaining accessible for educational and community-based projects.

The goal of PSCHI02 is not simply to provide technical support, but to create a practical learning environment where students can gain hands-on experience using industry-relevant tools and processes.

---

## Why osTicket Was Chosen

### Real-World Experience

One of the primary objectives of this project is to help students transition from classroom learning to real-world IT support operations.

Rather than building a custom ticketing system, osTicket allows students to work within a platform designed specifically for managing support requests, documenting solutions, and tracking service activities.

Students gain experience with workflows similar to those used in professional help desk and technical support environments.

---

### Industry-Relevant Skills

Through osTicket, students can develop practical experience in:

* Ticket creation and management
* Incident tracking
* Customer communication
* Ticket assignment and ownership
* Escalation procedures
* Knowledge base creation
* Documentation best practices
* Service desk operations

These skills are commonly required for positions such as:

* Help Desk Technician
* IT Support Specialist
* Desktop Support Technician
* Technical Support Analyst
* Systems Support Technician

---

### Open Source and Accessible

osTicket is an open-source platform, making it an excellent choice for educational and community-driven initiatives.

Because there are no licensing costs, students can learn on a production-style system without requiring expensive enterprise software.

This allows the project to focus resources on infrastructure, training, and hands-on experience rather than software licensing.

---

### Integration with Modern IT Infrastructure

The platform integrates seamlessly with the technology stack used throughout the PSCHI02 project, including:

* Rocky Linux
* Apache Web Server
* PHP
* MariaDB
* Grafana
* Prometheus

By deploying and maintaining the platform themselves, students gain exposure not only to ticket management but also to the supporting infrastructure that powers modern web applications.

---

### Learning Beyond Ticket Resolution

The project is designed to provide experience beyond simply answering support requests.

Participants are exposed to:

* Linux administration
* Server deployment
* Monitoring and observability
* Web application management
* Database administration
* Technical documentation
* Team collaboration
* Service management workflows

This creates opportunities to build both technical and professional skills that are directly applicable to entry-level IT roles.

---

## Supporting the PSCHI02 Mission

PSCHI02 was created to bridge the gap between learning IT concepts and applying them in a practical environment.

By using osTicket, students are able to interact with a professional support platform while building experience that can be discussed during interviews, added to resumes, documented on GitHub, and demonstrated to future employers.

The project emphasizes learning through participation, documentation, collaboration, and real-world problem solving.

For many students, this may be their first opportunity to work within a live support environment using the same style of tools and workflows found throughout the IT industry.
