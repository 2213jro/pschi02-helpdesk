# PSChi02 Project History

## Initial Concept

The project began as a way for CompTIA students to gain hands-on experience using a real ticketing platform outside of the classroom.

## Phase 1: Infrastructure

- Purchased domain
- Deployed DigitalOcean VPS
- Installed Rocky Linux
- Installed Apache, PHP, and MariaDB
- Installed osTicket

## Phase 2: Knowledge Base

- Created troubleshooting articles
- Added lab documentation
- Built onboarding guides

## Phase 3: Student Access

- Created student accounts
- Developed onboarding process
- Tested ticket workflows

## Phase 4: Tier 2 Support

- Recruited volunteers
- Defined escalation procedures
- Created support roles

## Phase 5: Monitoring

- Installed Prometheus
- Installed Grafana
- Began monitoring server health
