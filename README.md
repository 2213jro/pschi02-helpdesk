# PSChi02 Help Desk

## Overview

PSChi02 Help Desk is a student-to-student IT support platform created to help aspiring IT professionals gain real-world experience using industry-standard ticketing software.

The platform allows students to:

- Create and manage support tickets
- Practice troubleshooting procedures
- Develop documentation skills
- Build knowledge base articles
- Gain experience with ticket workflows
- Collaborate with Tier 2 support volunteers

## Technologies Used

- Rocky Linux
- Apache
- MariaDB
- PHP
- osTicket
- DigitalOcean
- Cloudflare

## Project Goals

The goal of this project is to bridge the gap between classroom learning and real-world IT support experience by providing a safe environment where students can practice skills used in professional help desk environments.
